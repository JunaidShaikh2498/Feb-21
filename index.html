There are 3 types of CSS : Inline,Internal,Extenal.
For inline , we write the css for the element in the same tag using style attribute
For Internal, we write the css for the element in the head tag using style tag and then use CSS selectors inside it
For Extenal, we write the css for the element in a seperate CSS file, use CSS selectors inside it and link that to the main html file
